# FitMeal-Prep-Assistant

# Project Overview
Developed an intelligent meal preparation and fitness planning application as part of the AWS AI Practitioner Challenge certificate program through Udacity. This project leverages Amazon Bedrock's foundation models to deliver personalized nutrition guidance, workout plans, and meal recommendations.

[![AWS PartyRock](https://img.shields.io/badge/AWS-PartyRock-orange?style=for-the-badge&logo=amazon)](https://partyrock.aws/u/Nouran02/YwB22hilj/FitMeal-Prep-Daily-Assistant)

# Key Features
• Personalized BMI analysis and ideal weight calculation based on user metrics
• AI-generated meal recommendations with macronutrient breakdowns
• Custom 7-day meal plans adapted to dietary preferences and cultural cuisine
• Comprehensive exercise programs tailored for gym or home workouts
• Location-aware recipe suggestions using locally available ingredients

# Impact
This application empowers users to achieve their fitness goals through evidence-based nutrition planning and accessible workout guidance, demonstrating practical applications of AI in health and wellness.
